# Tables
Para estilização das tabelas, existe uma classe que formata toda a tabela sem dificuldades.
Para tal, é necessário seguir a semantica descrita abaixo:

Necessário inseruir a classe `.base-table` para que seja totalmente formatada. E, para definir a cor do  header, basta usar uma das classes de thema:

Class                        | Cor           | Hexadecimal | Representação
:--------------------------- | :------------ | ----------- | ------------------------------------------------------------------:
`.theme-colored-gray`        | Cinza         | #656565     | ![img](../../static/css/gray.png)
`.theme-colored-red`         | Vermelha      | #D71F3C     | ![img](../../static/css/red.png)
`.theme-colored-purple`      | Roxo          | #641E75     | ![img](../../static/css/purple.png)
`.theme-colored-aquamarine`  | Aquamarine    | #20BEC6     | ![img](../../static/css/aquamarine.png)
`.theme-colored-green`       | Verde         | #20C67B     | ![img](../../static/css/green.png)
`.theme-colored-yellow`      | Amarelo       | #F2B838     | ![img](../../static/css/yellow.png)

É possível também alinhar a tabela ao centro da tela usando a classe `.central`.

E, é possível manter a tabela como header fixo na tela usando a claase `.fixed`.

## Implementação

```html
<table class="base-table central theme-colored-aquamarine">
    <thead>
      <tr>
        <th>Valor 1</th>
        <th>Valor 2</th>
        <th>Valor 3</th>
        <th>Total</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Descrição</td>
        <td>380,20</td>
        <td>950,00</td>
        <td>1.450,20</td>
      </tr>
      <tr>
        <td>Descrição</td>
        <td>380,20</td>
        <td>950,00</td>
        <td>1.450,20</td>
      </tr>
      <tr>
        <td>Descrição</td>
        <td>380,20</td>
        <td>950,00</td>
        <td>1.450,20</td>
      </tr>
      <tr>
        <td>Descrição</td>
        <td>380,20</td>
        <td>950,00</td>
        <td>1.450,20</td>
      </tr>
    </tbody>
    <tbody>
      <tr>
        <th>Total</th>
        <td>360,00</td>
        <td>1.140,60</td>
        <td>4.350,80</td>
      </tr>
    </tbody>
  </table>
</section>
```
